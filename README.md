你只需要关心你的具体业务逻辑实现就可以了,像对接普通接口一样可以轻松对接基于OPEN AI平台，提供统一标准接口规范。

开发计划

1. 支持三方应用接入如微信QQ飞书钉钉等
2. 支持联网，和必应GPT4搜索
3.  支持官方插件
4. 支持开发者开发插件
5. 官方PC聊天应用支持
6. 更多开放想法

4. 或者在github开源案例项目提issues
